Overview

The server version of UBUNTU 16.04 LTS is installed.

References

  • Official Ubuntu Documentation
    • UBUNTU 16.04 LTS Server Guide
      • Installation
        • Installing from CD

Procedure

The steps are documented in Installing from CD .

Step 1: Download Software

The ISO image is downloaded from here .

Step 2: Boot the System off the ISO Image

For an example of how to do this under XEN, see the build diary for AUBURN .

Step 3: Select Language

The following screen will then appear. English is already selected:

Press Enter to continue.

Step 4: Boot Menu

The following screen will then appear. Install Ubuntu Server is already selected:

Press Enter to continue.

Step 5A: Select Language (Again)

The following screen will then appear. English is already selected:

Press Enter to continue.

Step 5B: Select Location

The following screen will then appear.

Select Australia .

Press Enter to continue.

Step 6: Select Keyboard Layout

The following screen will then appear.

Select No .

Press Enter to continue.

The following screen will then appear. English (US) is already selected:

Press Enter to continue.

The following screen will then appear. English (US) is already selected:

Press Enter to continue.

Step 7: Discover Hardware Configuration

The following screen will then appear.

Step 8: Set Host Name

The following screen will then appear.

Enter auburn (or a different host name).

Select Continue .

Press Enter to continue.

Step 9: Create New User

The following screen will then appear.

Enter douglas (or a different user).

Select Continue .

Press Enter to continue.

The following screen will then appear. douglas is copied from the previous screen.

Select Continue .

Press Enter to continue.

The following screen will then appear.

Enter the required password for douglas .

Select Continue .

Press Enter to continue.

The following screen will then appear.

Enter the same password again for douglas .

Select Continue .

Press Enter to continue.

Step 10: Encrypt Home Directory

The following screen will then appear.

Select No .

Press Enter to continue.

Step 11: Configure Clock

The following screen will then appear.

Select New South Wales .

Press Enter to continue.

Step 12: Partition Disk

The following screen will then appear. Since there is only one (1) disk drive, this is already selected:

Press Enter to continue.

The following screen will then appear.

Select Yes to overwrite whole disk.

Press Enter to continue.

The following screen will then appear.

Select Continue .

Press Enter to continue.

The following screen will then appear.

Select Yes to overwrite whole disk.

Press Enter to continue.

Step 13: Install Base System

The following screen will then appear.

Step 14: How To Update System

The following screen will then appear.

Leave proxy information blank.

Select Continue .

Press Enter to continue.

The following screen will then appear.

The following screen will then appear. No automatic updates is already selected:

Step 15: Install Additional Packages

The following screen will then appear. Standard system utilities is already selected:

Select OpenSSH Server .

Select Continue .

Press Enter to continue.

The following screen will then appear.

Press Enter to continue.

Step 16: Finish the Installation

The following screen will then appear.

Select Yes .

Press Enter to continue.

The following screen will then appear.

The following screen will then appear.

The following screen will then appear.

Select Continue .

Press Enter to continue.

Note: It is not necessary to remove the CD-ROM before the reboot because the default boot order is disk then CD-ROM (see Setting boot order for domUs ).
