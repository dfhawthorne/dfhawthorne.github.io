Procedure

Installation Media

See 1.2 Obtaining and Preparing Installation Media for instrutions.

The installation media was the ISO image, V921569-01.iso , which was downloaded from Oracle Software Delivery Cloud .

Start Installation

See 2.1 Starting the Installation for instructions.

With the installation media in the CD-ROM drive, the server (or VM image) was started. The following screen appeared:

Option I was selected to install the operating system.

Choose Language

On the following screen, English (Australia) was chosen:

Continue is clicked.

Installation Summary

The following screen appeared:

The local time was already set to that of Sydney NSW based on UTC as the system time.

Set Installation Destination

See 2.4 Configuring the System Storage and Network for instructions.

Installation Destination was clicked to have the details filled out as follows:

Clicked Done to return to Installation Summary .

Network & Hostname

Back on the Installation Summary screen, Network & Host Name was clicked (this is below Installation Destination ).

After Configure… was selected for eth0 , the following screen was filled as follows to manually set the IP address:

Click Save to return to the Network & Host Name screen.

After Configure… was selected for eth1 , the following screen was filled as follows to use a local-link:

Click Save to return to the Network & Host Name screen.

Hostname was changed to redfern1.yaocm.id.au , and the Apply button was clicked to get the following screen:

Clicked Done to return to Installation Summary .
