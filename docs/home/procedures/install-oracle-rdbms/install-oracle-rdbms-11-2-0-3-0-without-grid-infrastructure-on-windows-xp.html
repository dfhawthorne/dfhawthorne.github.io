Overview

This procedure documents the installation of a new Oracle home for Personal Edition of Oracle RDBMS 11.2.0.3.0 on Windows XP SP3 computer (PENRITH).

Download Patchset

Downloaded the following files from My Oracle Support for 11.2.0.3.0 for Windows 32 bit:

  1. p10404530_112030_WINNT_1of6.zip
  2. p10404530_112030_WINNT_2of6.zip

Although there are six (6) files available for for download, I need just these two (2) for installation of the RDBMS software.

This is the latest patchset available for Windows 32-bit. There are no PSUs available.

Unzipped Files

I unzipped these files into I:\Software\RDBMS_11_2_0_3\ .
