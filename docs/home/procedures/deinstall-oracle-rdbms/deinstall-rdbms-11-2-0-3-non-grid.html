Overview

Since I had installed RDBMS 11.2.0.3 (See Upgrade REPOS to 11.2.0.3.3 from 11.1.0.7.0 on GRIDCTRL ) before I had installed Giid Infrastructure ( Install Grid Infrastructure 11.2.0.3.3 on GRIDCTRL , I had to de-install RDBMS 11.2.0.3.

Prequisite

RDBMS 11.2.0.3 installed

Procedure

Ran the following commands on GRIDCTRL :

export ORACLE_HOME=/opt/oracle/app/OracleHomes/db11.2.0.3
export ORACLE_BASE=/opt/oracle
cd $ORACLE_HOME/deinstall/
./deinstall

The log is attached as deinstall_11_2_0_3.log .
