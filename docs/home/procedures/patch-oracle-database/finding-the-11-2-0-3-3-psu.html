11.2.0.3 PatchSets for RDBMS

I was getting very frustrated trying to find the patchset to update the REPOS database on GRIDCTRL from 11.1.0.7 to 11.2.0.3.3.

Answer from Oracle Support

I raised a support call with My Oracle Support and got the following answer:

  • 11.2.0.3.0 PATCH SET FOR ORACLE DATABASE SERVER for Linux x86.
  • DATABASE PATCH SET UPDATE 11.2.0.3.3 (INCLUDES CPU JUL2012) for Linux x86.
  • How To Find RDBMS patchsets on My Oracle Support (Doc ID 438049.1)
  • Quick Reference to Patchset Patch Numbers (Doc ID 753736.1)
  • Important Changes to Oracle Database Patch Sets Starting With 11.2.0.2 (Doc ID 1189783.1)
  • Where do I find that on My Oracle Support (MOS) [Video] (Doc ID 1194734.1)

What I was doing Wrong

The critical thing I was missing was that I had not expanding the Patching Quick Links menu on the Patches & Updates tab. This I learnt from the first video in Where do I find that on My Oracle Support (MOS) [Video] (Doc ID 1194734.1) . Things are simple once you know how.
