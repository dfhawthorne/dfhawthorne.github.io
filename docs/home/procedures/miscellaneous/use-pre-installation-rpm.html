Overview

I have found the use of the Oracle Pre-Installation RPM to be very useful in many software installations.

References

  • 12.1 Grid Infrastructure Installation Guide
    • 3 Automatically Configuring Oracle Linux with Oracle Preinstallation RPM
      • 3.3 Installing Oracle Linux with Public Yum Repository Support

Procedure

After OEL 7 or OEL 6.5 is installed, I just run the following command:

sudo yum install oracle-rdbms-server-12cR1-preinstall

A sample log is attached as yum-install.log.
