Overview

With OEL7, the firewall changes from IPTABLES to FIREWALLD. This procedure shows how to disable the new firewall for home networks.

References

  • Michael Kerrisk - man7.org
    • systemctl
    • yum

Procedure

Since this is a home system, I decided to disable the firewall.

The commands were: 

systemctl stop firewalld.service
systemctl disable firewalld.service
