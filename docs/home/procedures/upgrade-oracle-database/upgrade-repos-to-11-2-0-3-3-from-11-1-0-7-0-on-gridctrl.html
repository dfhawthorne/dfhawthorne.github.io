Overview

This is a rough procedure I used to upgrade the REPOS database on GRIDCTRL from 11.1.0.7 to 11.2.0.3.3 prior to upgrading to EM12C R2.

I am following the advice from Oracle Enterprise Manager Cloud Control 12c Release 2 Installation on Oracle Linux 5.8 and 6.3 by Tim Hall , in which he says that the OMR should be at least 11.2.0.2.0.
