Overview

GORDON is a 64-bit Linux system that is used to host the 12C version of the OEM Repository (OMR) for the OEM Server (OMS) on CLOUDCTL when the OMS was upgraded to 13.1. The original OMR on GRIDCTRL was migrated to this server.

Planning

Because of memory limitations on VICTORIA , I decided to create a VM image on RYDE using VirtualBox. The settings were:

Setting Value
Memory (GB) 4
Number of CPUs 1
Disk Size (GB) 100
IP Address 192.168.1.107
