Overview

I destroyed the previous build of VICTORIA by replacing it with OVMS 3.2.6.

Procedure

Download Software

  1. Go to Oracle Software Delivery Cloud - Oracle Linux and Oracle VM
  2. Sign in using your Oracle account
  3. Accept the conditions
  4. Search using:
    5. Select a Product Pack Oracle VM
    Platform x86 64 bit
  5. Click Go
  6. Select Oracle VM 3.2.6 Media Pack for x86_64 (64 bit)
  7. Click Continue (at the bottom of the page)
  8. Download Oracle VM Server 3.2.6 for x86_64 (64 bit) (V40080-01)
  9. Burn this image onto a CD-ROM

Part number is B74784-01.

Install System

Insert the CD-ROM created above into the machine brought as New Server for Oracle 11.2 Clusterware and RAC Study .

These are the same parameters used in the first build of the OVM Server .

Issue with Startup of Victoria

After the reboot, the issue described in ETH0 has wrong MAC Address after Upgrade to OVMS 3.2.6 stopped all network communications. I had to change the MAC address for eth0 to get around the problem.
