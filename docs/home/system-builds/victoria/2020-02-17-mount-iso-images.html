Overview

In my quest to have automatic build of operating systems using KickStart, I need some way of mounting the ISO images as local files.

References

Procedure

Create Mount Points

Create the following mount points for each OEL ISO image as follows: 

mkdir /mnt/OEL_6_5_server_x86_64
mkdir /mnt/OEL_7_0_server_x86_64
mkdir /mnt/OEL_7_2_server_x86_64
mkdir /mnt/OEL_7_3_server_x86_64
mkdir /mnt/OEL_7_4_server_x86_64

Set Up Mounts on Reboot

In order to have the OEL ISO images automatically mounted after a reboot, I updated /etc/fstab as follows: 

cat >>/etc/fstab <<DONE
# -------------------------------------------------------------------------------
# ISO disks mounts
# -------------------------------------------------------------------------------
/OVS/iso_pool/V41362-01.iso  /mnt/OEL_6_5_server_x86_64 iso9660 ro,loop 0 0
#/OVS/iso_pool/V46135-01.iso  /mnt/OEL_7_0_server_x86_64 iso9660 ro,loop 0 0
#/OVS/iso_pool/V100082-01.iso /mnt/OEL_7_2_server_x86_64 iso9660 ro,loop 0 0
#/OVS/iso_pool/V834394-01.iso /mnt/OEL_7_3_server_x86_64 iso9660 ro,loop 0 0
/OVS/iso_pool/V921569-01.iso /mnt/OEL_7_4_server_x86_64 iso9660 ro,loop 0 0
DONE

Not all OEL ISO images are mounted because there are only eight (8) /dev/loop devices.
