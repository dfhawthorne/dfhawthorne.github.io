Overview

After finding that I could no longer run EM12C on a 32-bit machine along with an 11.2.0.4 database instance, I decided to remove the EM12C instance and software from GRIDCTRL .

Procedure

Following the procedure in 03 Deinstall EM12CR1 and Others , I ran the following commands: 

/opt/oracle/software/em12103p1/runInstaller -deinstall
