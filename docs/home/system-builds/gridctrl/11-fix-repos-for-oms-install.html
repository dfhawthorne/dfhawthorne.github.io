Overview

As part of the installation of OMS 12.1.0.3.1 on CRONULLA , several changes were made to the REPOS database.

Diagnosis

Clean Up after Removal of OMS

When I removed the OMS from GRIDCTRL , I overlooked the following items:

  • Public synonyms
  • Tablespaces
  • Users

Installation of OMS

When I installed the OMS on CRONULLA , I overlooked the database parameters that needed to be changed:

  • processes
  • session_cached_cursors
  • sga_max_size
  • sga_target
  • shared_pool_size
  • pga_aggregate_target
  • db_cache_size
