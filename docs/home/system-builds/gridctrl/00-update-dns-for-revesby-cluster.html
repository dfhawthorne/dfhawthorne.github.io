Overview

The build of the REVESBY cluster requires the allocation of several network names for the hosts and cluster. Also, the network names for previous builds of VMWare systems were removed as were decommissioned Windows boxes.

Procedure

Update Zone Files

On GRIDCTRL , the following files were updated:
  • /var/named/chroot/var/named/yaocm.id.au.zone
  • /var/named/chroot/var/named/yaocm.id.au.rr.zone

Restart BIND

service named restart

File Downloads

  1. yaocm.id.au.rr.zone
  2. yaocm.id.au.zone
