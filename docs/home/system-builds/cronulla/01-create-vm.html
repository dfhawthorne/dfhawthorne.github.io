2014-03-04 Create VM

Contents

  1. 1 Overview
  2. 2 References
  3. 3 Design
  4. 4 Procedure
    1. 4.1 Create the VM
    2. 4.2 Load the OS DVD
    3. 4.3 Installation

Overview

Now that I have reviewed the system requirements for installing OMS as described in 00 Design, I am ready to create the VM image.

References

  • Oracle® Linux Installation Guide for Release 6
    • Chapter 1 Pre-installation Configuration
      • 1.1 System Requirements

Design

The advice from 1.1 System Requirements seems to be suspect:

You can install Oracle Linux 6 on systems with the x86 or x86_64 architecture, a minimum of 1 GB of memory, a minimum of 1 GB of disk space, and an unlimited number of CPU cores.

Emphasis Mine

The VM should be created with the following:

Component Value
RAM 6GB
Disk 30GB
DVD V41362-01.iso

Procedure

Create the VM

The VM is created as follows:


Clicked Next.

As recommended in 00 Design, 6GB of RAM is allocated as follows:


Clicked Next.

The following hard disk drive is allocated:


Clicked Create.

Choose the default hard disk type:


Clicked Next.

Choose fixed size:


Clicked Next.

Added the 15GB as required from 00 Design to the default value of 12GB suggested and an extra 10% for safety to get a final allocation of 30GB:


Clicked Create.

Load the OS DVD

Set the DVD drive to read from the V41362-01.iso image.

Installation

Tried to get all the required packages installed at first, but I gave up after trying to guess where they would be. Installed the system as a Web Server.

Installed the OS without any special options.
