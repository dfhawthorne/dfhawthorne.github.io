Design

The advice from 1.1 System Requirements seems to be suspect:

You can install Oracle Linux 6 on systems with the x86 or x86_64 architecture, a minimum of 1 GB of memory, a minimum of 1 GB of disk space , and an unlimited number of CPU cores.

Emphasis Mine

The VM should be created with the following:

Component Value
RAM 6GB
Disk 30GB
DVD V41362-01.iso
