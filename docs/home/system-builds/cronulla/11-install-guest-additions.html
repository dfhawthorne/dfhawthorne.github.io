Overview

Installed Guest Additions into VM.

Procedure

Upgrade VirtualBox

I upgraded VirtualBox to VirtualBox-4.3.10-93012-Win .

Insert CD

From the VM Menu, select the following option:

Mount the CD

After logging in as root , I used the following command to mount the CD: 

mount -o ro -t iso9660 /dev/cdrom /mnt

Install Guest Additions

I ran the following command: 

/mnt/VBoxLinuxAdditions.run
