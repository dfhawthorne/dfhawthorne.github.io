Overview

The OVM Template for Cloud Control 12.1.0.3.0 needs to be assembled and customised before starting.

References

Readme for Media Pack B75437-01

Procedure

Unzip the Downloaded Files

Run the following commands on VICTORIA to unzip all of the downloaded files:

cd /OVS/seed_pool/OVM_EM12R3
unzip V41319-01.zip
unzip V41320-01.zip
unzip V41321-01.zip
unzip V41322-01.zip
unzip V41323-01.zip

Extract the Files

Run the following commands on VICTORIA to extract the OVM files:

cat *.tgz|tar xzvf -
