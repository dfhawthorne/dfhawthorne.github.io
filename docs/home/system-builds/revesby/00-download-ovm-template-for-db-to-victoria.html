Overview

I used the new OVM server created on VICTOIA to host the downloaded template of 12C RAC database.

References

Readme for Media Pack B74025-01 Oracle VM Templates for Oracle Database Media Pack v1 for x86 (64 bit)

Procedure

Download Software

  1. Go to Oracle Software Delivery Cloud - Oracle Linux and Oracle VM
  2. Sign in using your Oracle account
  3. Accept the conditions
  4. Search using:
    5. Select a Product Pack Oracle VM Template
    Platform x86 64 bit
  5. Click Go
  6. Select Oracle VM Templates for Oracle Database Media Pack for x86 (64 bit)
  7. Click Continue (at the bottom of the page)
  8. Download Oracle VM Templates for Oracle Database - Single Instance and RAC 12cR1 (12.1.0.1.0) - Oracle Linux (64 bit) (part 1 of 2) (V38984-01)
  9. Download Oracle VM Templates for Oracle Database - Single Instance and RAC 12cR1 (12.1.0.1.0) - Oracle Linux (64 bit) (part 2 of 2) (V38985-01)

Part number is B74025-01.

Upload the Files to VICTORIA

On the Windows 8.1 system, run the following commands under MobaXTerm :

sftp root@192.168.1.100
cd /OVS/scratch
mput V3898*
exit
