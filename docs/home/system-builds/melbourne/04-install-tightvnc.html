Procedure

4.4.2.1. Simple Installation (includes database if necessary) says that:

Please note that you need to install tightvnc-java on this computer to access a virtual machine's console.

Install Software

I used the following procedure from MOS Document 728618.1 (as root ): 

rpm -Uvh http://oss.oracle.com/oraclevm/manager/RPMS/tightvnc-java-1.3.9-3.noarch.rpm
