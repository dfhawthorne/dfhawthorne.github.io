Design

Hardware

Based on 4.3.1. Hardware Requirements , I plan to set up a VM under VirtualBox with the following parameters:

Items Value
Memory 8 GB
Processor Type 64 bit
Number of Processors 1
Swap Space 2.1 GB
Hard Disk Space (Including Swap) 25 GB

Software

I plan to use OEL 6.5 (V41362-1) for the base software. This is supported according to 4.3.2.1. Operating Systems .
