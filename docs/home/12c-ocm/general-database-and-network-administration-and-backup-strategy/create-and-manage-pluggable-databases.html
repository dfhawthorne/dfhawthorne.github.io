Procedure

Use DBCA to Create a PDB

The simplest procedure is to use DBCA as described in Managing PDBs in a CDB using DBCA . My investigation of this procedure can be found in “ Create PDB using DBCA (Simple) ”.

Manual Procedures for Creating a PDB

In 38.1.1 Techniques for Creating a PDB , the following ways to create a PDB are listed:

  1. Create a PDB by using the seed
  2. Create a PDB by cloning an existing PDB or non-CDB
  3. Create a PDB by plugging an unplugged PDB into a CDB. I demonstrated this in Clone PDB from Unplugged PDB .
  4. Create a PDB by using a non-CDB

These procedures are described in “ Create and configure a PDB ” as part of the 12 OCP Upgrade Exam. I also investigate SNAPSHOT COPY clones here.
