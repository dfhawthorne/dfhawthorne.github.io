Overview

The exam topic is “ Create and manage database configuration files ”. There are two (2) main database configuration files:

  1. Initialization Parameter File
  2. Server Parameter File

The Initialization Parameter File is maintained through a text editor, and requires an instance restart to take effect.

The Server Parameter File is maintained through two (2) means:

  1. The ALTER SYSTEM SET command under SQL*Plus
  2. Oracle Enterprise Manager (OEM)
