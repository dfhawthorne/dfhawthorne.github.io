Setup

Three systems are involved:

  1. CRONULLA

Prepare for Conversion

Navigate to the Data Guard Administration page where I have the following:


Since there is only one (1) standby database defined, there is no need to make a selection.

Click Convert .

Enter Host Credentials

Ensure that the correct host credentials are entered for where the physical standby database instance is running. In this case, that host is BOTANY :


Click Continue .

Conversion Confirmation

The following screen asks for confirmation to convert the physical standby to a snapshot standby:


Click Yes .

Conversion Complete

Note that the role has changed to Snapshot Standby .
