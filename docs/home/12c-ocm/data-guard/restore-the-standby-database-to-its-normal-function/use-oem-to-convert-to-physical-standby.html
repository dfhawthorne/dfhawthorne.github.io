Summary

The snapshot standby can be restored to its normal function as a physical standby through Oracle Enterprise Manager (OEM)

Procedure

Setup

Three systems are involved:

  1. CRONULLA
  2. PADTSOW
  3. BOTANY

Prepare for Conversion

Navigate to the Data Guard Administration page where I have the following:


Since there is only one (1) standby database defined, there is no need to make a selection.

Click Convert .

Enter Host Credentials

Ensure that the correct host credentials are entered for where the snapshot standby database instance is running. In this case, that host is BOTANY :


Click Continue .

Conversion Confirmation

The following screen asks for confirmation to convert the snapshot standby to a physical standby:


Click Yes .

Processing: Convert Standby Database

The following progress screen appears:


Conversion Complete

When the conversion has completed successfully, the main data guard administration screen reappears with a message as follows:


Note that the role has changed to Physical Standby .
