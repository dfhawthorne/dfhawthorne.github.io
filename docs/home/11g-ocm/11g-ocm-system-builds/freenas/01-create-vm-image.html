Using VMWare® Workstation,

  1. Choose Custom .
  2. Click Next .
  3. Hardware compatibility is Workstation 8.0 .
  4. Click Next .
  5. Choose I will install the operating system later in order to avoid the Easy Install option.
  6. Click Next .
  7. Guest operating system is Other .
  8. Version is FreeBSD .
  9. Click Next .
  10. Virtual machine name is FreeNAS .
  11. Click Next .
  12. Leave the number of processors at 1 with 1 core per processor.
  13. Click Next .
  14. Change the memory to 1024MB.
  15. Click Next .
  16. Change Network connection to Use bridged networking because the VM image needs its own IP address.
  17. Click Next .
  18. Leave I/O controller types as they are.
  19. Click Next .
  20. Create a new virtual disk.
  21. Click Next .
  22. Leave Virtual disk type as is.
  23. Click Next .
  24. Maximum disk size is 8.0GB.
  25. Allocate all disk space now.
  26. Store virtual disk as a single file.
  27. Click Next .
  28. The name of the Disk File is FREENAS_root.vmdk .
  29. Click Next .
  30. Click Customize Hardware… .
  31. Add one (1) Network Adapter that use a Host-only network connection. This is for the private networks for the NAS access from clients.
  32. Go back to Ready to Create Virtual Machine .
  33. Click Finish .
  34. Update settings to add one (1) SCSI Disk Drive of 20GB. The name of the Disk File is FeeNAS_PENRITH.vmdk .
