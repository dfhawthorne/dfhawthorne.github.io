Overview

This is the anchor document for DNS changes made to the GRIDCTRL server.

The original procedure is at http://yaocm.wordpress.com/2009/12/27/setup-dns-on-gridctrl/ .

/etc/named.conf

The procedure for setting up a BIND Configuration File is at 18.2. /etc/named.conf of the RHEL 5.0 Deployment Guide .

The configuration file is named.conf in /var/named/chroot/etc .

Zone Files

The procedure for setting up a Zone File is at 18.3. Zone Files of the RHEL 5.0 Deployment Guide .

The file is uploaded as yaocm.id.au.zone in /var/named/chroot/var/named .

Reverse Name Resolution Zone Files

The procedure for setting up a Reverse Name Resolution Zone File is at 18.3.4. Reverse Name Resolution Zone Files of the RHEL 5.0 Deployment Guide .

The file is uploaded as yaocm.id.au.rr.zone in /var/named/chroot/var/named .
