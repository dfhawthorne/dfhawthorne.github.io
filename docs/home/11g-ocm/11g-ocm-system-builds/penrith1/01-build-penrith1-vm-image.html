Using VMWare® Workstation,

  1. Choose Custom .
  2. Click Next .
  3. Hardware compatibility is Workstation 8.0 .
  4. Click Next .
  5. Choose I will install the operating system later in order to avoid the Easy Install option.
  6. Click Next .
  7. Guest operating system is Linux .
  8. Version is Red Hat Enterprise Linux 5 .
  9. Click Next .
  10. Virtual machine name is PENRITH1 .
  11. Click Next .
  12. Leave the number of processors at 1 with 1 core per processor.
  13. Click Next .
  14. Leave the memory at 1024MB.
  15. Click Next .
  16. Change Network connection to Use bridged networking because the VM image needs its own IP address.
  17. Click Next .
  18. Leave I/O controller types as they are.
  19. Click Next .
  20. Create a new virtual disk.
  21. Click Next .
  22. Leave Virtual disk type as is.
  23. Click Next .
  24. Maximum disk size is 8.0GB.
  25. Allocate all disk space now.
  26. Store virtual disk as a single file.
  27. Click Next .
  28. The name of the Disk File is PENRITH1_root.vmdk .
  29. Click Next .
  30. Click Customize Hardware… .
  31. Add two (2) Network Adapter s that use a Host-only network connection. This is for the private networks for the interconnect and NAS access.
  32. Add one (1) SCSI Disk Drive of 8GB. The name of the Disk File is PENRITH1_U01.vmdk .
  33. Go back to Ready to Create Virtual Machine .
  34. Click Finish .
  35. Update settings to add an extra 12GB disk drive called PENRITH1_U01.vmdk .
