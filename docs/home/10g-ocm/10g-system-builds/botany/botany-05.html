Thursday 10 July, 2008 - 18:00

The problem with the bridged connection as described in Botany (04) has been solved by adding a D-Link 100/10 Ethernet card to the Lenovo XP machine. This was similar to the solution I used in setting up the Grid Control on the Lenovo Linux machine.

It appear that the in-built Gigabit Ethernet card on the Lenovo ThinkCentre A55 is not compatible with RHEL 4.2 and RHEL 3.7 (with and without VMware).
