Missing Targets

Sunday 27 July, 2008 - 19:19

The ASM instance and its associated listener were missing from the list of targets on the botany1 node in the Grid Control definitions. 

$ORA_AGENT_HOME/bin/agentca -d - c botany1,botany2

This command stops the OMS agent and discovered all available targets on the cluster defined by nodes botany1 and botany2. (This latter part seems to be unnecessary because the nodes are listed in the OCR.)
