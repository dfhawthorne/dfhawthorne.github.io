Summary

Network Command Centre has six (6) main services:

  1. Inter-Region Latency
  2. Network Path Analyzer
  3. Network Visualizer
  4. Virtual Test Access Points
  5. VCN Flow Logs
  6. Capture filters

Network Path Analyzer, VCN Flow Logs, Capture Filters, and Virtual Test Access Points need configuration before use.

Inter-region Latency and Network Visualizer are available as dashboards from the Network Command Center.
