Summary

Network Visualizer is accessible from the Network Command Center. There are three (3) levels of topology maps:

  1. Regional: Gateways
  2. VCN: Subnets, VLANs, and gateways to other resources
  3. Subnet: Instances, load balancers, FSS, and OKE clusters
