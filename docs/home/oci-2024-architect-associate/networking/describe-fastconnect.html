Summary

A FastConnect is a dedicated connection between customer premise and an OCI data centre.

FastConnect Overview

FastConnect Overview says:

Oracle Cloud Infrastructure FastConnect provides an easy way to create a dedicated, private connection between your data center and Oracle Cloud Infrastructure. FastConnect provides higher-bandwidth options, and a more reliable and consistent networking experience compared to internet-based connections.

There are two (2) types of peering:

  1. Private extends on-premises network to OCI using a physical connection.
  2. Public allows access to public-facing resources in OCI without traffic passing the Internet.

Connectivity Models

There are three (3) models for connecting to OCI using FastConnect:

  1. Oracle Partners have an existing relationship with Oracle that allows the partner to terminate network connections within OCI data centres.
  2. Third-Party Provider require an letter of authority (LOA) to enter the OCI data centre in order to terminate a network link there by placing network equipment and links.
  3. Colocation is where the customer provides their own equipment and link into an OCI data centre. A LOA is required for data center access. This works best if the customer already has a presence at the data centre.
