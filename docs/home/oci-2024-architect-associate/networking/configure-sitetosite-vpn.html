Site-to-Site VPN

Offers a simple and secure way to connect corporate network to OCI over an existing Internet connection.

Can be a redundant connection for FastConnect.

Connects CPE (Customer Premises Equipment) to DRG (Dynamic Routing Gateway) via two (2) redundant IPsec tunnels.

Need to create the following in OCI:

  1. Dynamic Routing Gateway (DRG)
  2. Customer Premise Equipment (CPE) representing the router in the customer's network.
