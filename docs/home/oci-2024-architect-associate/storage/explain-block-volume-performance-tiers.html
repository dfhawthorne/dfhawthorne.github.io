Summary

There are four (4) performance levels for block volumes. The performance level of a block volume can be changed at any time.

Block Volume Performance

Virtual Performance Unit (VPU) is a measure of the I/O performance for a block volume that encompasses the following limits:

  • IOPS per GB
  • Max IOPS per Volume
  • Size for Max IOPS (GB)
  • KBPS per GB
  • Max MBPS per Volume

Performance levels are:

  • Ultra High Performance: Can purchase 30 - 120 VPUs per GB/month. Only available for block volumes.
  • Higher Performance: Purchase 20 VPUs per GB/month
  • Balanced: Default to 10 VPUs per GB/month
  • Lower Cost: incurs only storage cost - no additional VPU cost. Only available for block volumes.
