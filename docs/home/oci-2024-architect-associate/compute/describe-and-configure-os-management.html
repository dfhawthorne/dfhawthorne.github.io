Introduction to OS Management Service

  • Automated patch management
  • Simplified package management
  • Integrated service with OCI console
  • Common Vulnerabilities and Exposures Lookup
  • Free service for OCI subscribers

Oracle Cloud Management Service

Steps to enable Oracle Cloud Management Service (OSMS) are:

  1. Create an OCI Dynamic Group
  2. Set up IAM policies for OS Management
  3. Ensure Oracle Cloud Agent software is installed and running on the instance
  4. Ensure that the OSMS Agent plugin is enabled
