Types of Compute Offerings

There are two (2) types of OCI compute offerings:

  1. Bare metal
  2. Virtual machine

Bare metal hosts are dedicated to a single tenancy. Bare metal can be deployed either as a whole machine or a dedicated VM server which is not shared with any other customer.
