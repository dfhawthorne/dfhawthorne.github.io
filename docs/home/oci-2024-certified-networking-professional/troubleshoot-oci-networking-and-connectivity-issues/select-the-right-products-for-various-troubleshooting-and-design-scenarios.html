Network Command Centre

Visualisation and dashboard services

Network Visualiser
The network visualiser provides a diagram of the implemented topology of all VCNs in a selected region and tenancy.
Inter-Region Latency
The Inter-Region Latency dashboard provides the average network round-trip latency (round-trip time or RTT) for all pairs of regions in the OCI realm.

Troubleshooting and Diagnostic Services

Network Path Analyser
NPA provides a unified and intuitive tool to identify virtuak network configuration issues that impact connectivity.

Traffic Monitoring Services

VTAPs
A virtual test access point (VTAP) mirrors all traffic to or from a designated source and send it to a selected target in order to facilitate troubleshooting, security analysis, and data monitoring.
Capture Filters
A capture filters contains a set of rules governing what traffic is captured by a VTAP or flow log.
Flow Logs
Use flow log configuration to capture network traffic telemetry to support monitoring and security needs.
