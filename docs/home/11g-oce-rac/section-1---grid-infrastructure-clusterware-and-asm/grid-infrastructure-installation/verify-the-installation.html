In Oracle® Grid Infrastructure Installation Guide 11g Release 2 (11.2) for Linux , the section, 4.4 Confirming Oracle Clusterware Function , says to run the following command as root :

# crsctl check crs

CRS-4638: Oracle High Availability Services is online
CRS-4537: Cluster Ready Services is online
CRS-4529: Cluster Synchronization Services is online
CRS-4533: Event Manager is online

In Oracle® Grid Infrastructure Installation Guide 11g Release 2 (11.2) for Linux , the section, 4.5 Confirming Oracle ASM Function for Oracle Clusterware Files , says to run the following command as grid , if the OCR and voting disks are installed on ASM disks:

$ srvctl status asm
ASM is running on node1,node2
