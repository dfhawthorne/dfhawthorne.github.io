Using ThinkServer EasySetup

First boot failed because I did not have the CD inserted in time. After a few minutes, the message came up saying that there was no operating system installed. 

Error 1962: No operating system found.Press any key to repeat boot sequence.

I inserted the CD called ThinkServer EasyStartup and pressed the space bar to reboot.

Selected the following options:

  • Language: English
  • Keyboard layout: English

The next screen advised me that, if I wanted to create a RAID array in Onboard SATA RAID mode, I would have to do the following:

  1. Exit the EasyStartup program
  2. Set the SATA mode as RAID in the Setup Utility program
  3. Restart the server
  4. Press <ctrl>+I when prompted to enter the RAID configuration utility program.

I was going to set up RAID-1 but I decided against because this is a test server for my educational needs. Nearly all of the data can be thrown away.

Accepted the license agreement.

Set the date and time

I continued to the main interface.

I chose Install Operating System from the main menu. The type of O/S was Other . I then found I had to install the operating system myself.

Install Oracle VM Server

I inserted the Oracle VM Server 3.1.1 disc 1 CD and rebooted. Tested the CD. No problems found.
